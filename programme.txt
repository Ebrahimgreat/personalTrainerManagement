Programme Name: Hybrid Workout Programme
Programme Description: This programme is designed for hypertophy purposes. Each muscle group is trained 3 times a week.

Template : Push

  Exercise: Machine Chest Press, Set: 3, Reps: 10-12
  Exercise: Tricep Pushdown, Set: 3, Reps: 10-12
  Exercise: Chest Dips, Set: 3, Reps: 10-12
  Exercise: Cable Chest Fly, Set: 3, Reps: 10-12

  Exercise: Reverse Fly, Set: 2, Reps: 10-12
  Exercise: Barbell Shrugs, Set: 3, Reps: 10-12
  Exercise: Cable Row, Set: 3, Reps: 10-12
